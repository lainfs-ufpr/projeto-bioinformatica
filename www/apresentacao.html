Controle de Qualidade de Sequências FASTQ

Este projeto foi desenvolvido pela diretoria de Bioinformática da LAINFS, para facilitar a análise de qualidade de dados transcriptômicos, especialmente arquivos do tipo FASTQ.

Aqui você pode:

  • Upload de arquivos FASTQ individuais ou em lote
  • Explorar múltiplos tipos de gráficos
  • Selecionar paletas de cores personalizadas para os gráficos
  • Exportar seus resultados para análise externa
